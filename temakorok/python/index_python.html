Mi is az a Python?

A Python napjainkban az egyik legismertebb programozási nyelv. Guido van Rossum hozta létre ezt a nyelvet 1991-ben.

Mire használjuk:

  • Web szerkeztés/fejlesztés
  • Szoftver fejlesztés
  • Matematikai egyenletek/feladatok megoldása
  • Rendszer írás
